lhie1规则生成

### 简介
由于规则的种类太多，之前的生成器并不能很好的地工作了，加上我也基本不用其他规则，所以把之前的脚本改为lhie1的规则生成脚本。目前提供如下功能：
1. 导入和筛选节点
2. 定义Auto节点
3. 去广告规则开关
4. TestFlight开关

> 导入节点可以识别二维码和托管

> 如果你和我一样没有自定义规则和需要关闭去广告规则，那这个脚本和workflow已经没区别了
